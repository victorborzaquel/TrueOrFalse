# True Or False

## React Native App

Aplicativo feito para quem quer entender como a comparação do JavaScript funciona.  
Basta digitar o valor, escolher o tipo de dado e o operador que deseja usar na comparação.

## Tecnologias

- Select Dropdown
- React Native
- Styled Components
- Typescript
